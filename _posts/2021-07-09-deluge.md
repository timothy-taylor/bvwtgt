My current synthesizer performances are based on a conceptual workflow revolving around the Synthstrom Deluge playing double duty as a looper and sequencer with the main sound coming from a modular synth. I've found I always need to open the Deluge manual more than I should, so I'm going to actually write some stuff down here to see if I can remember the exact key combinations:

> *order of key presses matters*

> in a blank project; in song mode;

> `pad` + `<select>`: turns that row in an audio track

> `<learn>` + `pad`; select your audio in, `[Left.]` for mono in with monitoring

> hold `<record>` to check whether the individual track is armed to record, but it always is by default

> press `<record>` to arm recording; 

> press `<play>` to start recording; press `<launch>` on that row to stop recording; this first loop sets your loop length

> to begin sequencing, go into a new track ABOVE the looping track. Set to CV and sequence as desired; you can set a kit to cv by holding `<section>` and pressing `<cv>`

> return to song mode at any point to overdub with new tracks by pressing `<record>` + `<pad>` for a track directly BELOW the most recent loop. Once gain pressing `<launch>` on that track to stop recording.

> all of these individual overdubs are displayed as tracks and have all the normal Deluge functionality: mute, timestretch, effects, detune, etc
